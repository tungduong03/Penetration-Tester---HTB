# Enumeration

Enumeration chính là quá trình:

- Thu thập thông tin.
- Hiểu hệ thống đang chạy gì.
- Tìm các điểm yếu tiềm năng.
- Xác định vector tấn công.

Hai loại thông tin quan trọng cần tìm

## 1. Chức năng hoặc tài nguyên có thể tương tác

Ví dụ:

- Web login page
- FTP server
- SMB share
- SSH service
- API endpoint
- Upload function

Những thứ này có thể trở thành điểm tấn công trực tiếp.

## 2. Thông tin dẫn đến thông tin khác

Ví dụ:

- Username
- Email
- Domain name
- Version phần mềm
- Banner
- Config file
- Source code

Thông tin nhỏ có thể dẫn đến lỗ hổng lớn hơn.
