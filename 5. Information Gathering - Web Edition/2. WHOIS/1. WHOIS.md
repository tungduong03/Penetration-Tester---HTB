# WHOIS

```bash
reDrose18@htb[/htb]$ whois inlanefreight.com

[...]
Domain Name: inlanefreight.com
Registry Domain ID: 2420436757_DOMAIN_COM-VRSN
Registrar WHOIS Server: whois.registrar.amazon
Registrar URL: https://registrar.amazon.com
Updated Date: 2023-07-03T01:11:15Z
Creation Date: 2019-08-05T22:43:09Z
[...]
```

Mỗi bản ghi WHOIS thường chứa các thông tin sau:

- Domain Name: The domain name itself (e.g., example.com)

- Registrar: Công ty đăng kí tên miền (e.g., GoDaddy, Namecheap)

- Registrant Contact: Người hoặc tổ chức đăng kí tên miền

- Administrative Contact: Người chịu trách nhiệm quản lý tên miền.

- Technical Contact: Người phụ trách các vấn đề kỹ thuật liên quan đến domain này.

- Creation and Expiration Dates: Thời điểm tên miền được đăng ký và thời điểm hết hạn của nó.

- Name Servers: Các máy chủ chuyển đổi tên miền thành địa chỉ IP.
