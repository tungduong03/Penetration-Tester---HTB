# Digging DNS

## DNS Tools

| Công cụ           | Chức năng chính                  |
| ----------------- | -------------------------------- |
| dig               | Truy vấn DNS mạnh mẽ và chi tiết |
| nslookup          | Truy vấn DNS cơ bản              |
| host              | Truy vấn DNS nhanh gọn           |
| dnsenum           | Enumeration DNS tự động          |
| fierce            | Tìm subdomain và DNS Recon       |
| dnsrecon          | Enumeration DNS toàn diện        |
| theHarvester      | Thu thập OSINT, email, DNS       |
| Online DNS Lookup | Website tra cứu DNS trực tuyến   |


dig
- Công cụ DNS mạnh nhất.
- Hỗ trợ hầu hết các loại record.
- Phân tích DNS chuyên sâu.

nslookup
- Dễ dùng.
- Kiểm tra nhanh A, AAAA, MX.

host
- Kết quả ngắn gọn.
- Phù hợp kiểm tra nhanh.

dnsenum
- Enumeration tự động.
- Brute-force subdomain.
- Zone Transfer.

fierce
- Tìm subdomain.
- Recursive enumeration.
- Phát hiện wildcard DNS.

dnsrecon
- Kết hợp nhiều kỹ thuật DNS Recon.
- Thu thập thông tin toàn diện.

theHarvester
- Công cụ OSINT.
- Thu thập: Email, Nhân viên, DNS Information

Online Lookup
- Dùng trên trình duyệt.
- Phù hợp khi không có terminal.

Các lệnh DIG 
```bash
dig domain.com
dig domain.com A
dig domain.com AAAA
dig domain.com MX
dig domain.com NS
dig domain.com TXT
dig domain.com SOA
dig +trace domain.com
dig -x IP
dig +short domain.com
dig +noall +answer domain.com
```
