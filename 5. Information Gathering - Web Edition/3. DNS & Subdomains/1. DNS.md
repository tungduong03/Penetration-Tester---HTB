# DNS

## How DNS Works


| Khái niệm                 | Ý nghĩa                                 |
| ------------------------- | --------------------------------------- |
| Domain Name               | Tên miền dễ đọc cho con người           |
| IP Address                | Địa chỉ số của thiết bị trên Internet   |
| DNS Resolver              | Máy chủ chuyển tên miền thành IP        |
| Root Name Server          | Máy chủ DNS cấp cao nhất                |
| TLD Name Server           | DNS quản lý các TLD như `.com`, `.org`  |
| Authoritative Name Server | DNS chứa dữ liệu thật của domain        |
| DNS Records               | Các bản ghi DNS lưu thông tin khác nhau |


Chuỗi phân giải DNS 

- Domain
   
   ↓
- DNS Resolver
   
   ↓
- Root Server
   
   ↓
- TLD Server
   
   ↓
- Authoritative DNS
   
   ↓
- IP Address


Các loại record 

| Record | Mục đích trong Recon                  |
| ------ | ------------------------------------- |
| A      | Tìm IP máy chủ                        |
| AAAA   | Tìm IPv6                              |
| CNAME  | Tìm alias và hạ tầng liên quan        |
| MX     | Xác định hệ thống email               |
| NS     | Xác định DNS/Hosting Provider         |
| TXT    | Thu thập thông tin dịch vụ bên thứ ba |
| SOA    | Thu thập thông tin quản trị DNS       |
| SRV    | Tìm dịch vụ đặc biệt                  |
| PTR    | Reverse DNS Lookup                    |
