# FTP 

FTP sử dụng 2 kết nối TCP:

TCP Port 21: Kênh điều khiển (Control Channel), dùng để gửi lệnh.

TCP Port 20: Kênh dữ liệu (Data Channel), dùng để truyền file.

Nếu quá trình truyền file bị gián đoạn, FTP có thể tiếp tục truyền sau khi kết nối được khôi phục.

# TFTP (Trivial File Transfer Protocol)

TFTP là phiên bản đơn giản hơn FTP, chỉ dùng để truyền file.

Đặc điểm

- Sử dụng UDP (không phải TCP) nên không đảm bảo độ tin cậy.
- Không yêu cầu xác thực (không username/password).
- Không hỗ trợ:
    - Liệt kê thư mục (ls).
    - Quản lý file/thư mục.
    - Các tính năng bảo mật.

Do thiếu bảo mật, TFTP thường chỉ được sử dụng trong mạng nội bộ (LAN) hoặc môi trường đáng tin cậy.

## Default Configuration

vsFTPd (Very Secure FTP Daemon) là một trong những FTP Server phổ biến nhất trên Linux

### Install vsFTPd

```bash 
reDrose18@htb[/htb]$ sudo apt install vsftpd
```

File config `/etc/vsftpd.conf`

Các cấu hình mặc định quan trọng

| Cấu hình                   | Ý nghĩa                                                             |
| -------------------------- | ------------------------------------------------------------------- |
| `listen=NO`                | Không chạy độc lập, chạy thông qua `inetd` (hoặc dịch vụ tương tự). |
| `listen_ipv6=YES`          | Lắng nghe kết nối IPv6.                                             |
| `anonymous_enable=NO`      | **Không cho phép Anonymous FTP** (mặc định an toàn hơn).            |
| `local_enable=YES`         | Cho phép tài khoản Linux đăng nhập FTP.                             |
| `dirmessage_enable=YES`    | Hiển thị thông báo khi người dùng vào thư mục.                      |
| `use_localtime=YES`        | Sử dụng giờ hệ thống.                                               |
| `xferlog_enable=YES`       | Ghi log các lần upload/download.                                    |
| `connect_from_port_20=YES` | Data connection dùng TCP Port 20 (Active FTP).                      |
| `secure_chroot_dir=/var/run/vsftpd/empty`    | Thư mục chroot bảo mật.                                             |
| `pam_service_name=vsftpd`  | Xác thực người dùng thông qua PAM.                                  |
| `ssl_enable=NO`            | **Không bật SSL/TLS**, dữ liệu truyền dưới dạng **plain text**.     |


File `/etc/ftpusers`

```bash
reDrose18@htb[/htb]$ cat /etc/ftpusers
```

Đây là danh sách các tài khoản bị cấm đăng nhập FTP.

Nếu các tài khoản này tồn tại trên Linux thì vẫn không thể đăng nhập FTP, vì bị chặn bởi file này.

# Dangerous setting

FTP có thể được cấu hình cho phép người dùng ẩn danh (anonymous) truy cập mà không cần tài khoản.

| Cấu hình                      | Ý nghĩa                                |
| ----------------------------- | -------------------------------------- |
| `anonymous_enable=YES`        | Cho phép đăng nhập Anonymous           |
| `anon_upload_enable=YES`      | Cho phép Anonymous upload file         |
| `anon_mkdir_write_enable=YES` | Cho phép Anonymous tạo thư mục         |
| `no_anon_password=YES`        | Không yêu cầu mật khẩu Anonymous       |
| `anon_root=/home/user/ftp`    | Thư mục gốc của Anonymous              |
| `write_enable=YES`            | Cho phép ghi (upload, xóa, đổi tên...) |

Nếu bật cả anonymous_enable và write_enable, đây là một cấu hình rất nguy hiểm vì bất kỳ ai cũng có thể ghi file lên FTP.

| **Cấu hình**              | **Mô tả**                                                                                                                      |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------------ |
| `dirmessage_enable=YES`   | Hiển thị thông báo khi người dùng lần đầu truy cập vào một thư mục mới.                                                        |

| `chown_username=username` | Chỉ định tài khoản sẽ trở thành **owner** của các file do người dùng anonymous upload.                                         |
| `local_enable=YES`        | Cho phép các tài khoản người dùng cục bộ (local users) đăng nhập vào FTP.                                                      |
| `chroot_local_user=YES`   | Giới hạn (chroot) người dùng cục bộ trong thư mục Home của họ, không thể truy cập ra ngoài.                                    |
| `chroot_list_enable=YES`  | Sử dụng danh sách các người dùng sẽ bị áp dụng chính sách **chroot** (hoặc được miễn chroot tùy cấu hình `chroot_local_user`). |
| `hide_ids=YES`            | Ẩn thông tin UID/GID và tên người dùng/nhóm trong kết quả `ls`, thay vào đó hiển thị là **ftp**.                               |
| `ls_recurse_enable=YES`   | Cho phép sử dụng lệnh `ls -R` để liệt kê toàn bộ cây thư mục (recursive listing).                                              |
